How This Works

📄

1. Document Ingestion

Resume, project docs, and technical notes are split into small, meaningful chunks and converted into vector embeddings.

🔍

2. Semantic Retrieval

Your question is embedded and compared against all document chunks using FAISS to find the most relevant passages.

🤖

3. Grounded Generation

Gemini 2.5 Flash generates an answer using only the retrieved context — never inventing facts or skills.

🛡️

4. Hallucination Guard

A strict system prompt ensures the AI will explicitly say "I don't have that information" rather than fabricate claims.
